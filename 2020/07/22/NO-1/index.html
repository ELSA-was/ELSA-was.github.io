NO.1 搭建个人博客

Posted on 2020-07-22 Edited on 2020-07-23

准备工作

  1. 安装Nodejs

  2. 下载npm

  3. 下载cnpm

前置步骤

  1. 安装Nodejs

  2. 查看node版本
    node -v

  3. 查看npm版本[注1]
    npm -v

  4. 安装淘宝的cnpm管理器
    npm install -g cnpm --registry=http://registry.npm.taobao.org

  5. 查看cnpm版本
    cnpm -v

搭建论坛

  1. 安装hexo框架
    cnpm install -g hexo-cli

  2. 查看hexo版本
    hexo -v

  3. 创建blog目录
    mkdir blog

  4. 进入blog目录
    cd blog

  5. 生成博客&初始化博客
    sudo hexo init

  6. 启动本地博客服务
    hexo s

论坛操作

  1. 本地访问地址
    http://localhost:4000/

  2. 创建新的文章
    hexo n "文章标题"

  3. 清理日志
    hexo clean

  4. 生成
    hexo g

  5. Github创建一个新的仓库
    YourGithubName.github.io

  6. 在blog目录下安装git部署插件
    cnpm install --save hexo-deployer-git

  7. 配置_config.yml 

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
      type: git
      repo: https://github.com/YourGithubName/YourGithubName.github.io.git
      branch: master

  1. 部署到Github仓库里
    hexo d

  2. 访问博客的地址
    https://YourGithubName.github.io/

  3. 下载yilia主题到本地
    git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
    修改hexo根目录下的 _config.yml 文件 ： theme: yilia
